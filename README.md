## 快速开始（恢复环境）

如需在新设备上恢复 Blank 的环境：

```bash
# 1. 克隆仓库
git clone https://github.com/opriz/blank-evolution.git
cd blank-evolution

# 2. 设置环境变量
export EMAIL_AUTH_CODE='你的邮箱授权码'
export GITHUB_TOKEN='你的GitHub令牌'

# 3. 运行恢复脚本
bash scripts/restore.sh
```

详细恢复指南：[docs/RECOVERY.md](docs/RECOVERY.md)
